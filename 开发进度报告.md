## 💡 使用说明

### 本地开发
```bash
# 1. 进入后端目录
cd backend

# 2. 安装依赖
npm install

# 3. 配置环境变量
cp .env.example .env
# 编辑 .env 文件

# 4. 创建数据库
mysql -u root -p -e "CREATE DATABASE wordheat CHARACTER SET utf8mb4;"

# 5. 导入表结构
mysql -u root -p wordheat < database/schema.sql

# 6. 导入词库
node database/scripts/import_words.js --file=./data/pep_primary_words.json

# 7. 启动服务
npm start
```

### 小程序预览
使用微信开发者工具打开 `frontend` 目录即可。
