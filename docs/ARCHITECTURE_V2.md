### 5. API 接口规范

统一响应格式:
```json
{
  "success": true,
  "data": {},
  "message": "操作成功"
}
```

核心接口:
- `GET /api/v1/words/today` - 今日学习任务
- `POST /api/v1/words/answer` - 提交答案
- `GET /api/v1/users/stats` - 学习统计
- `POST /api/v1/auth/wx-login` - 微信登录

### 6. 数据库设计 (简化版)

保留核心表:
- `users` - 用户表
- `word_banks` - 词库表
- `learning_records` - 学习记录表
- `user_statistics` - 用户统计表

### 7. 艾宾浩斯算法 (已存在，验证通过)

复习间隔: [1, 2, 4, 7, 15, 30, 90] 天
熟悉度更新: 答对+10，答错-5
掌握条件: 熟悉度≥80 且 连续答对3次
